Google Apps Script – Formular erstellen

🔧 So verwendest du das Script:

  1. Öffne script.google.com und erstelle ein neues Projekt.
  2. Lösche den vorhandenen Code und füge das Script oben ein.
  3. Klicke auf ▶ Ausführen → Funktion: erstelleTest
  4. Erteile bei der Berechtigungsanfrage Zugriff auf Google Forms.
  5. Das fertige Formular erscheint in deinem Google Drive.
  6. Wichtig: Notiere die angezeigte FormID – du brauchst sie für das Sheets-Script!
ℹ️ Generiere zuerst einen Test unter „Einstellungen".
📊
Google Apps Script – Sheets Auswertung

📊 So richtest du die Auswertung ein:

  1. Öffne das Google Sheet, das automatisch mit dem Formular verknüpft wurde (Formular → Antworten → In Tabelle öffnen).
  2. Klicke auf Erweiterungen → Apps Script.
  3. Füge das Auswertungs-Script ein und passe die FORM_ID an (aus Schritt 6 des Forms-Scripts).
  4. Führe einmalig auswertungEinrichten aus – das erstellt das Auswertungsblatt und alle Formeln.
  5. Optional: Führe triggerEinrichten aus – dann wird die Auswertung automatisch bei jeder neuen Antwort aktualisiert.

💡 Das Auswertungsblatt enthält: Punkte je Schüler:in, automatische Note (Ö-Skala), Klassendurchschnitt, Notenspiegel-Diagramm und farbliche Hervorhebung.
