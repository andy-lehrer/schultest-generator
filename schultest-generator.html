Google Apps Script – Formular erstellen
So verwendest du das Script:
  1. script.google.com öffnen → Neues Projekt
  2. Vorhandenen Code löschen → Script einfügen
  3. ▶ Ausführen → Funktion: erstelleTest
  4. Berechtigungen erteilen
  5. Form-ID aus dem Dialog notieren!
ℹ️ Generiere zuerst einen Test unter „Einstellungen".

Google Apps Script – Sheets Auswertung
So richtest du die Auswertung ein:
  1. Google Sheet der Formular-Antworten öffnen
  2. Erweiterungen → Apps Script
  3. Script einfügen und FORM_ID eintragen
  4. auswertungEinrichten ausführen
  5. Optional: triggerEinrichten für Auto-Aktualisierung
